# Precedence of Python Operators

| Operator                              | Description                        |
|---------------------------------------|------------------------------------|
| `() [] {}`                            | 元组, 列表, 字典, 集合             |
| `[m:n] f() .`                         | 下标, 切片, 函数调用, 属性引用     |
| `**`                                  | 指数 (最高优先级)                  |
| `~ + -`                               | 按位翻转, 一元正号和负号           |
| `* / % //`                            | 乘, 除, 取模和取整除               |
| `+ -`                                 | 加, 减                             |
| `>> <<`                               | 右移, 左移                         |
| `&`                                   | 按位与                             |
| `^ <bar> `                            | 按位异或, 按位或                   |
| `<= < > >= == != is is not in not in` | 比较运算符, 身份运算符, 成员运算符 |
| `= %= /= //= -= += *= **=`            | 赋值运算符                         |
| `not`                                 | NOT运算符                          |
| `and`                                 | AND运算符                          |
| `or`                                  | OR运算符                           |
