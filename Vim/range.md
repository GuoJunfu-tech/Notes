# Range

m[,n]

| Symbol      | Address                                             |
|-------------|-----------------------------------------------------|
| n           | 文件的第n行                                         |
| $           | 文件的最后一行                                      |
| 0           | 虚拟行, 文件第一行的上方                            |
| .           | 光标所在行                                          |
| 'm          | 包含位置标记m的行                                   |
| '<          | 高亮选区的起始行                                    |
| '>          | 高亮选区的结束行                                    |
| %           | 整个文件(1,$的缩写)                                 |
| {address}-n | 所给地址的上方n行, 如果n忽略, 则默认为1             |
| {address}+n | 所给地址的下方n行, 如果地址忽略, 则默认为光标所在行 |
| /pattern/   | 用模式指定范围                                      |
