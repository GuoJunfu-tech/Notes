# Window
* `CTRL-w_r` 调整窗口的位置, 循环调整
* `CTRL-w_R` 调整窗口的位置, 反向循环调整
* `CTRL-w_x` 交换当前窗口和下一个窗口
* `CTRL-w_HJKL` 调整窗口的位置, 指定上下左右
* `CTRL-w_hjkl` 在上下左右分割窗口, 并将光标置于对应窗口(自定义)
* `CTRL-w__` 垂直方向最大化当前窗口
* `CTRL-w_|` 水平方向最大化当前窗口
* `CTRL-w_=` 使各窗口均等
* `CTRL-w_<>` 左右调整窗口大小, 自定义为`ALT-,.`
* `CTRL-w_+-` 上下调整窗口大小, 自定义为`ALT-=-`
